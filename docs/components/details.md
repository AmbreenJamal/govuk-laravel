# Details

Show an expandable dropdown with optional information.

```html
<x-govuk::details
    label="Show more details"
>
    // More content here...
</x-govuk::details>
```

## Props

| Name  | Type   | Default  | Description |
| ----- | ------ | -------- | ----------- |
| label | string | Required | A title that describes what is written inside the box |
