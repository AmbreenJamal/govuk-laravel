# Grid Row

Create a row for layout as per the GOV.UK layouts style.

```html
<x-govuk::grid-row>
    <x-govuk::grid-column width="two-thirds">
        // Content...
    </x-govuk::grid-column>
    
    <x-govuk::grid-column width="one-third">
        // Content...
    </x-govuk::grid-column>
</x-govuk::grid-row>
```

A `grid-row` should be wrap one or more `grid-column` elements.
