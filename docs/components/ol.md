# Ordered List

Create an `ol` element, with optional spacing.

```html
<x-govuk::ol
    spaced
>
    <li>My list item</li>
    // More list items...
</x-govuk::ol>
```

## Props

| Name     | Type | Default | Description |
| -------- | ---- | ------- | ----------- |
| spaced   | bool | false   | Whether list items should have extra space |
