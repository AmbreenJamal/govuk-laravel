# H3

Create a first-level heading. 

```html
<x-govuk::h3
    size="l"
>
    My Heading
</x-govuk::h3>
```

## Props

| Name | Type   | Default | Description |
| ---- | ------ | ------- | ----------- |
| size | string | l       | How large the heading should be |
